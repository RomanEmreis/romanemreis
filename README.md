My work focuses on reducing that friction:

## Systems

### [Volga: composable Rust Web Framework](https://github.com/RomanEmreis/volga)

Bringing simplicity to Rust without sacrificing performance or architectural control.

Why it exists:

Rust web development often forces tradeoffs between ergonomics and power.  
Volga explores an alternative approach where explicit APIs preserve reasoning clarity without increasing complexity.

### [Neva: MCP SDK for AI agents and integrations](https://github.com/RomanEmreis/neva)

Model Context Protocol SDK designed to make AI agent systems production-ready.

Current AI tooling often optimises for experimentation rather than system design.  
Neva focuses on:

- Structured agent architecture
- Strong typing
- Extremely low runtime overhead
- Real-world integration scenarios
